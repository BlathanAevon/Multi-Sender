# Tabi Multi Sender


### Simple script to use Tabi testnet


# How to use:

Put private keys in `configs/wallets.txt`
<br>
Put proxies in format `http://user:password@host:port` in `configs/proxies.txt`

### On Linux / MacOS:
**Run**
`./scripts/run.sh`
    
### On Windows:
**Run**
`./scripts/run.bat`
